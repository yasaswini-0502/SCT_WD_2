# ⏱ Stopwatch & Countdown Web App
A feature-rich stopwatch and countdown timer web application built with HTML, CSS, and JavaScript.
This project provides a clean, responsive, and interactive interface for tracking time, recording laps, and running countdowns with multiple customization options

🚀 Features

▶ Start / Pause stopwatch or countdown seamlessly
🔄 Reset to start fresh anytime
🏁 Lap Timer – track multiple intervals while stopwatch is running
🌙 Dark Mode Toggle for a modern look
⏱ Countdown Timer with preset options (1, 2, 3 minutes)
🔔 Time’s Up Alert when countdown finishes
📱 Responsive UI for both desktop and mobile

🛠️ Tech Stack
HTML5 – structure of the app
CSS3 – styling, animations, and dark mode
JavaScript (ES6) – stopwatch, countdown, and interactivity

📂 Installation & Usage

# Clone the repository
git clone https://github.com/your-username/stopwatch.git

# Navigate into the project folder
cd stopwatch

# Open index.html in your browser
